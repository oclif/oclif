## `oclif manifest [PATH]`

Generates plugin manifest json (oclif.manifest.json).

```
USAGE
  $ oclif manifest [PATH] [--jit]

ARGUMENTS
  PATH  [default: .] Path to plugin.

FLAGS
  --[no-]jit  Append commands from JIT plugins in manifest.

DESCRIPTION
  Generates plugin manifest json (oclif.manifest.json).
```
